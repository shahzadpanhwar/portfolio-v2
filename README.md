# Portfolio V2

A modern responsive portfolio website built using HTML, CSS, and JavaScript.

## 🚀 Live Demo

https://shahzadpanhwar.github.io/portfolio-v2/

## ✨ Features

- Responsive Design
- Modern UI
- Typing Animation
- Animated Counters
- Scroll Reveal Animations
- Responsive Hamburger Menu
- Projects Section
- Resume Download
- Contact Section
- Back to Top Button

## 🛠️ Technologies

- HTML5
- CSS3
- JavaScript

## 📂 Installation

Clone the repository:

```bash
git clone https://github.com/shahzadpanhwar/portfolio-v2.git
```

Open `index.html` in your browser.
